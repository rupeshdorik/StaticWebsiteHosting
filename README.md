# AWS Static Website Hosting using Python SDK (boto3)

## Project Overview

This project automates the deployment of a static website to Amazon S3 using Python and the AWS SDK (boto3).

## Technologies Used

- Python
- boto3
- Amazon S3
- AWS CLI
- HTML
- CSS
- JavaScript

## Features

- Automated website deployment
- Static website hosting
- Responsive UI
- Secure AWS authentication
- Python-based file upload

## Project Structure

StaticWebsiteHosting/
├── website/
│ ├── index.html
│ ├── style.css
│ └── script.js
├── upload.py
├── requirements.txt
└── README.md

## How to Run

1. Configure AWS CLI:
   aws configure

2. Install dependencies:
   pip install -r requirements.txt

3. Run:
   python upload.py
