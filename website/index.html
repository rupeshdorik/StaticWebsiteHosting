Static Website Hosting Using Python SDK (boto3)

Deploy websites automatically on Amazon S3 using Python, AWS SDK (boto3), and AWS CLI. This project demonstrates cloud automation, static website hosting, and secure deployment.

About This Project

This project demonstrates how to automate the deployment of a static website using Python, AWS SDK (boto3), Amazon S3, and AWS CLI without manually uploading files.

Automation

Website files are uploaded automatically using a Python script powered by boto3.

Cloud Hosting

Amazon S3 hosts the website as a highly available static website.

Secure Access

AWS IAM credentials authenticate the Python application securely before deployment.
